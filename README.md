# Legal Document Search & Clause Finder

## Objective
A web-based legal document search engine using TF-IDF ranking and Boolean retrieval.

## Tools & Technologies
- Python 3, Flask
- PyPDF2, python-docx
- HTML5, CSS3, JavaScript

## Installation
pip install flask PyPDF2 python-docx

## How to Run
1. python setup_templates.py
2. python app.py
3. Open http://localhost:5000

## Features
- Upload PDF, DOCX, TXT documents
- Boolean search (AND, OR, NOT)
- Inverted Index, TF, IDF, TF-IDF, Cosine Similarity, Ranking tables
